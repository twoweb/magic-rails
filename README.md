magic-rails
===================

For ruby ~> 1.9.3

magic.css 
https://github.com/miniMAC/magic


Installation
=======

Add this extension to your Gemfile:

```ruby
gem "magic-rails"
```

Then run:

```
bundle install
```

Add application.css.sass or application.css.scss (Ruby on Rails):

```
@import magic.css
```
