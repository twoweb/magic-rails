magic-rails
===================

For ruby ~> 1.9.3' 

magic.css for rails.
https://github.com/twoweb/magic-rails


Installation
=======

Add this extension to your Gemfile:

```ruby
gem "magic-rails"
```

Then run:

```
bundle install
```

Add application.css (Ruby on Rails):

```
@import magic.css
```
